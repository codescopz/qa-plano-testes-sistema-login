# Projeto QA - Sistema de Login

## Sobre o Projeto
Este projeto simula a documentação completa de testes para um sistema de Login e Recuperação de Senha.

## Objetivo
Garantir a qualidade, segurança e funcionamento correto do módulo de autenticação.

## Estrutura do Projeto
- Plano de Testes
- Casos de Teste
- Matriz de Rastreabilidade
- Cenários BDD

## Tipos de Testes Aplicados
- Testes Funcionais
- Testes Negativos
- Testes de Segurança
- Testes de Usabilidade
- BDD

---

 Projeto desenvolvido para evolução profissional na área de Quality Assurance.
