# Matriz de Rastreabilidade

## Projeto: Sistema de Login

A matriz de rastreabilidade garante que todos os requisitos do sistema estejam devidamente cobertos por casos de teste.

---

| Requisito | Descrição | Caso(s) de Teste |
|-----------|------------|------------------|
| RF01 | Usuário deve autenticar com e-mail e senha válidos | CT01 |
| RF02 | Sistema deve exibir erro para senha incorreta | CT02 |
| RF03 | Sistema deve bloquear conta após 5 tentativas inválidas | CT06 |
| RF04 | Sistema deve permitir recuperação de senha via e-mail | CT07, CT08 |
| RF05 | Campos obrigatórios devem ser validados | CT03, CT04, CT05 |

---

##  Análise de Cobertura

Total de Requisitos: 5  
Requisitos Cobertos: 5  
Cobertura: 100%
